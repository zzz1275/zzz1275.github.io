#### 自定义上下页
#vitepress #config

- [上下页配置参数](https://vitepress.dev/zh/reference/default-theme-prev-next-links)

在`.vitepress/config.js`中, 可以自定义上下页的显示
```js
export default defineConfig({
	themeConfig: {
		docFooter: {
			prev: '上一页',
			next: '下一页'
		}
	}
})

export default defineConfig({
	themeConfig: {
		docFooter: {
			prev: false,
			next: false
		}
	}
})
```
